# Hulatang Promotional Presentation

## Overview
An interactive HTML presentation for Hulatang (胡辣汤), created using reveal.js framework. This presentation showcases the traditional Chinese spicy soup with modern, responsive design and smooth animations.

## Features
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive Navigation**: Keyboard arrows, mouse clicks, and touch support
- **Smooth Animations**: Fade-ins, hover effects, and transitions
- **Modern Typography**: Montserrat for English, Noto Sans CJK for Chinese
- **Brand Colors**: Warm reds, browns, and oranges reflecting the soup's nature
- **13 Content Slides**: Complete coverage from introduction to call-to-action

## How to Use

### Opening the Presentation
1. Open `hulatang_ppt.html` in any modern web browser
2. The presentation will start automatically on the title slide

### Navigation Controls
- **Arrow Keys**: Navigate between slides (← → for previous/next)
- **Enter Key**: Move to next slide
- **Mouse Click**: Advance to next slide
- **Touch**: Swipe left/right on touch devices
- **ESC Key**: Toggle overview mode (see all slides at once)

### Presentation Features
- **Progress Bar**: Shows presentation progress at the bottom
- **Navigation Arrows**: Appears on hover for easy navigation
- **Responsive Layout**: Automatically adjusts to screen size
- **Interactive Elements**: Hover effects on cards and buttons
- **Smooth Transitions**: Professional slide animations

## Content Structure
1. **Title Slide**: Eye-catching introduction with Chinese characters

## Troubleshooting
### Common Issues
1. **Slides not advancing**: Check if JavaScript is enabled
2. **Fonts not loading**: Verify internet connection for Google Fonts
3. **Layout broken**: Try refreshing the page or clearing cache
4. **Animations not smooth**: Update browser to latest version
