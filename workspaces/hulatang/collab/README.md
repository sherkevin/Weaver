# 胡辣汤宣传PPT使用说明

## 文件说明
- `hulatang_ppt.html` - 完整的HTML演示文稿文件

## 如何打开
1. 直接双击 `hulatang_ppt.html` 文件，会在默认浏览器中打开
2. 或者在浏览器中右键选择"打开方式" → 选择您的浏览器

## 操作说明

### 基本导航
- **下一页**：点击"下一页"按钮，或按键盘右箭头键(→)，或按空格键
- **上一页**：点击"上一页"按钮，或按键盘左箭头键(←)
- **页面指示器**：右上角显示当前页码/总页数

### 移动端操作
- **左滑**：切换到上一页
- **右滑**：切换到下一页

### 特殊功能
- **第3页（食材展示）**：鼠标悬停在食材卡片上会有放大效果
- **第4页（制作过程）**：步骤会自动依次显示动画效果
- **第8页（顾客评价）**：评价内容会自动轮播展示

## 页面内容
1. **封面** - 品牌展示和核心卖点
2. **历史介绍** - 胡辣汤的起源和传承
3. **食材展示** - 精选食材介绍
4. **制作过程** - 动态展示制作步骤
5. **秘制配方** - 香料配比介绍
6. **八小时慢熬** - 时间工艺展示
7. **特色优势** - 核心竞争力总结
8. **顾客评价** - 好评轮播展示
9. **优惠活动** - 促销信息展示
10. **联系方式** - 店铺信息和营业时间

## 技术特性
- 响应式设计，适配电脑、平板、手机
- 流畅的页面切换动画
- 丰富的视觉效果和交互
- 无需外部依赖，可直接运行
- 支持键盘和触摸操作

## 浏览器兼容性
推荐使用以下浏览器的最新版本：
- Chrome
- Firefox
- Safari
- Edge

## 注意事项
- 确保浏览器支持JavaScript
- 建议全屏浏览以获得最佳体验
- 移动端建议横屏查看
